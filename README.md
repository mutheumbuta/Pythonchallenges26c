# Python_Practice_Challenges 
## Overview

This repository contains daily Python practice challenges and assignments designed to strengthen fundamental programming skills. The exercises cover essential Python concepts including strings, tuples, dictionaries, sets, object-oriented programming (OOP), and regular expressions (regex).

The goal of this project is to build strong problem-solving skills and practical coding experience through consistent practice.

## Skills Covered

Python Strings

Tuples

Sets

Dictionaries

Object-Oriented Programming (OOP)

Regular Expressions (Regex)

## Objectives
Strengthen Python fundamentals

Improve logical thinking and coding efficiency

Build a strong portfolio of Python exercises
